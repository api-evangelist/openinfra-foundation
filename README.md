# OpenInfra Foundation (openinfra-foundation)
The OpenInfra Foundation, which recently joined the Linux Foundation, is home to open infrastructure projects including OpenStack, Kata Containers, and StarlingX. It supports the development and adoption of open infrastructure for cloud computing, edge computing, and container technology.
